# RNClinic

אב-טיפוס ראשוני לאפליקציה לניהול קליניקה להשתלת שיער.

## מה יש כרגע
- דף MVP סטטי המציג יומן השתלות, צוות, כספים ומעקב שעות.
- נתוני דוגמה לצורך הדגמה בלבד.

## איך מריצים מקומית
### דרישות התקנה
- Python 3 (כולל מודול `http.server` שמגיע מובנה עם Python).
- דפדפן מודרני (Chrome / Edge / Firefox).

```bash
cd /path/to/RNClinic
python3 -m http.server 8000
```
ואז לפתוח בדפדפן: <http://localhost:8000>

ב-Windows אפשר גם:
```bash
cd C:\path\to\RNClinic
py -m http.server 8000
```

### הרצה ב-Windows עם קובץ Batch
הקובץ `run_rnclinic.bat` מפעיל:
- שרת UI על פורט 8000.
- שרת API מתוך תיקיית `server`.
- פתיחת Chrome על כתובת האפליקציה.

כדי להריץ, פתחו CMD בתיקיית הפרויקט והקלידו:
```bat
run_rnclinic.bat
```

## מטרות המוצר
- ניהול יומן השתלות לפי מטופל, צוות, חדר ותאריך.
- ניהול צוות (רופאים, אחיות, טכנאים) כולל שכר לפי שעה.
- ניהול עלויות והכנסות לפי טיפול, ספקים וחומרים מתכלים.
- מעקב אחר שעות עבודה ודוחות שכר.

## מודולים מרכזיים
### יומן השתלות
- יצירת השתלה חדשה עם פרטי מטופל, תאריך, משך, חדר, צוות ואביזרים.
- קישור להשתלה לחיובים/הכנסות.
- סטטוס: מתוכנן, בביצוע, הסתיים, בוטל.

### צוות ושעות
- כרטיס עובד: פרטים, תפקיד, שכר לשעה/משכורת.
- דיווח שעות ידני או באמצעות שעון נוכחות.
- חישוב שעות לפי תקופה, כולל חריגות.

### כספים
- קטלוג שירותים ועלויות חומרים.
- מעקב אחרי תשלומים, הנחות וזיכויים.
- דוחות רווחיות לפי חודש/רופא/שירות.

## נתוני ליבה מוצעים
- מטופל: מזהה, פרטי קשר, סטטוס.
- השתלה: מזהה, תאריך, משך, סטטוס, עלות/מחיר.
- עובד: מזהה, תפקיד, שכר.
- דיווח שעות: עובד, תאריך, שעות.
- חיוב: מטופל, סכום, סטטוס תשלום.

## שלבים מוצעים להמשך
1. אפיון UI למסכי יומן, צוות וכספים.
2. החלטה על טכנולוגיה (Web/Mobile) ואחסון נתונים.
3. MVP עם CRUD בסיסי ודוחות חודשיים.
